Please read the following instructions carefully.

This task is about reasoning - how people come to conclusions from arguments. On the following pages there are 16 reasoning problems that you will be asked to answer. In each problem you are given two statements together with a conclusion which may or may not follow from the statements. Your task in each case is to decide whether or not the conclusion has to be true, assuming that the two statements are true.

Each problem is about an imaginary letter-number pair. The first line is a rule that determines which letters may be paired with which numbers. An example rule would be:
If the letter is B then the number is not 7.

In each problem you must assume that the rule is true and then combine it with the information given in the second line. This will be about either the letter or the number of an imaginary pair, for example:
The letter is Y.
The number is not 4.

If the information concerns the letter then the conclusion will concern the number and vice-versa.

A full problem looks something like this:
If the letter is X then the number is 1.
The letter is X.
Conclusion: The number is 1.
Does the conclusion necessarily follow from the premises? Yes or No.

If you think the conclusion necessarily follows (if it must be true when the first two lines are true) then please select the YES box, otherwise tick the NO box. Please work through the problems in order and make sure you do not miss any. Do not return to a problem once you have finished and moved on to another.

The first problem will display when you click the button below.

Good luck!
